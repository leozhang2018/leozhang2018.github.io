CSS3 target 伪类实现 Tab 标签切换

发表于 2015-05-25

:target 是 CSS3 中新增的一个伪类,用以匹配当前页面的 URI 中某个标志符的目标元素。简单来说,当前页面 URL 下添加 #news 就会定位到 id=“news” 的位置，使用 :target 伪类可以像 :hover 等伪类一样对目标元素定义样式。
