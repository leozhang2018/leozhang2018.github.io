我在用什么

发表于 2018-05-07


时代在发展，工具在进步。

硬件篇

  • MacBook Pro (15-inch, 2017)
  • iMac 5K Retina
  • OnePlus 3 吃奥利奥
  • Mix2 体验全面屏
  • ATH-M50X BK
  • ATH LS50

影音

  • MPV + youGet + youtube-dl + Aria2 + 七牛云存储 + Filerun 一套私人流媒体组合拳
  • IINA

开发和效率

  • VScode Atom Sublime 按使用度排序，字体选择 Monaco，统一主题为 Dracula
  • Terminal 为 iTerm2 + oh-my-zsh 组合
  • NVM 进行 Node 版本管理
  • Homebrew 进行包管理，在不使用 Apple DNS 时通过 Mas 更快的下载 App Store 应用
  • 默认使用 VScode 内置版本管理插件
  • Alfred, 常用的 Workflow 包括：Dash 进行文本搜索、Terminal Finder 进行 Finder 和 Terminal 的相互打开、httpstat 可视化 curl 命令
  • 使用 Mackup 配合 Dropbox 管理环境配置

设计与原型

  • Avocode 进行设计稿的标注和切图（目前为止个人用的最佳标注工具）
  • Sip 作为全局取色工具

流程

  • 使用 Xmind 画流程图，其实更偏爱新的 Xmind Zen

备份与存储

  • 使用 Dropbox 进行重要数据同步与备份
  • 硬件上通过 Seagate Backup Plus 不定期备份

API 与 网络调试

  • 以前用 Postman，现在是 Paw 进行请求模拟
  • Charles HTTP 层抓包，必要时会搬出 Wireshark

记录

  • Waka Time 配合 VScode Atom Sublime

文档

  • 使用石墨文档进行一些文档的管理和分享
  • 使用 Typora 作为常用的 Markdown 编辑客户端

密码

  • 1Password
  • 在 Android 端，我选择使用 Msecure 更轻量

碎片化阅读

  • Instapaper 老用户
  • 使用 Feedly 订阅 RSS 流
  • Hacknews

IM 与社交

  • WeChat + Telegram + Tim
  • 心情好时会用 Path
  • Instagram
  • Google+ 不定期去 Linux community 看看热闹

    信息流

  • IFTTT

    科学上网

  • 目前使用 Bandwagonhost 性价比最佳，早些年用过 DigitalOcean、Vultr 线路质量目前在国内的网络环境下非常不乐观
  • 推荐家科学上网套餐提供商：优越加速（截止至 2018/5/7）
  • 云梯 RIP，包老板（kgen）现在应该还没出来

Chrome Extensions

  • Momentum
  • Vue DevTools 调试 Vue 应用必备
  • Redux DevTools 调试 React 应用必备
  • Proxy SwitchyOmega 快捷地管理和切换多个代理设置，可实现在线页面本地调试
  • 1Password 1Password 的扩展
  • Tampermonkey 油猴脚本，插件化浏览器必备
  • Instapaper
  • oneTab 快速整理标签页，减轻标签页混乱
  • Wappalyzer 快速识别网站用了哪些技术
  • Vimium 在浏览器里使用键盘操作
  • Emmet Review 模拟不同 Viewport 方便调试移动端网页
